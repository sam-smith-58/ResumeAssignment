Featured Projects

View selected projects below.

Fierce Spirit

For my Computer Design Tech course, we were tasked with creating a soap brand based on a phrase we chose. I went with "Fierce Spirit, focusing on a fire theme. After developing the logo, I went on to do a realistic mockup.

My favorite part of this piece was creating the marble flame for the logo. Using the twirl tool in Illustrator, I created many versions of this product in different color schemes. I use this marble effect for other projects too because of how fun it is.

Puppy Poke

In Graphic Design II, I was asked to design a dog treat brand that represented summer vacation. I came up with the brand, Puppy Poke, with Hawaii in mind.

I love illustrating, however I don't often get to illustrate for myself. This project gave me a lot of freedom to create something fun for my audience, as well as myself.

The Three-Eyed Wigglebeast

My Technical Illustration course included a project about designing stickers for a made-up company that sent monster stickers monthly. This project taught me many new skills about adding texture, outlines, and layers to my designs.

Skil Saw Isometric Drawing

Also from my Technical Illustration class, this was my final project of the course. It involved creating a realistic rendition of a table saw (the product that I chose) that could be used in a manual.

This project taught me how to apply textures to 3D objects and how to render them realistically.
